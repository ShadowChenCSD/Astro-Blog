# 1. 配置入口

​		选择`Manage Jenkins`，再选择`Manage Credentials`，准备配置凭据。

![管理密钥](管理密钥.jpg)

# 2. 凭据选择

选择`Jenkins`

![Jenkins](Jenkins.jpg)

选择`全局凭据`

![全局凭据](全局凭据.jpg)

# 3. 凭据添加

​		选择`Add Credentials`，**填入相关信息，保存**即可。

![增加凭据](增加凭据.jpg)
